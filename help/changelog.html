Initial version 1.0

  • Completly new code, a full working feature rich gamemode
  • Highly configurable & customizable
  • Custom hud and scoreboard in different styles ( improvments planned )
  • Damagescale system to balance damage done by weapons
  • Easy to use teamsystem with team management features
  • Integrated money system with two currencies to balance the too powerfull stuff easy
  • Powerfull shop system for weapons/entites and everything else ( cars and more )
  • Builtin vip system ( using gmod group system )
  • Integrated prop protection ( supports third-party pp systems if you need more features )
  • Easy ally system to share stuff with other players
  • Anitrdm protection ( only players related to raids can damage others and their props )
  • Basewars basic entites ( Printer, radar, turrets and much more )
  • Lifesystem for all entites and props ( Everything is destructible )
  • Doorsystem for basewars ( Buy/sell/destroy )
  • Unique raid system with configurable options ( raidtime/lastlife and more )
  • Generator/Powersystem for entities
  • Simple Drug system ( boost various attributes )
  • Flexible level & experience system
  • Configurable upgrade system for all entites & weapos ( players can upgrade their stuff ingame )
  • Modification system for custom entites
  • Chat commands and a gui quickmenu for the most common commands
  • Supports custom weapon packs ( Example config files for M9K & FA:S2 are included )
  • Support for darkrp printer and entites ( will expand in the future )
  • Compatible with pointshop and many other addons
  • And much more...
